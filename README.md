
# HomeAway

A website for students want to do masters abroad 



## Run Locally

Clone the project

```bash
  https://github.com/rufevean/FlyAway
```

Go to the project directory

```bash
  cd code
```

Install dependencies

```bash
  npm install
```
```
uncomment the earth component in home page and destination page ( ctrl + l)
```
Start the server

```bash
  npm run dev
```
